How to hide desktop icons before screen sharing on Mac

A small, local way to clean up the desktop before a call, demo, tutorial, or recording.

Why hide desktop icons?

Desktop icons can expose filenames, client names, screenshots, downloads, or just visual clutter during a screen share. Many people manually move files or switch desktops before a call, but that turns a tiny recurring problem into a ritual.

The simple option

CleanScreen Drop is a free Mac menu bar app that hides desktop icons before screen sharing, demo recording, or calls, then restores them afterward.

Open the app, use the menu bar item labeled Desk, choose hide before the call, and show again when you are done.

What it does not do

CleanScreen Drop is not a file cleaner, security product, encryption tool, or screen privacy shield. It does not read, move, upload, delete, index, or scan desktop files.

It only changes Finder's desktop icon visibility setting. There is no account, analytics, tracking, ads, file scanning, or network sync.

When it is useful

It is useful before Zoom calls, Google Meet calls, client screen shares, product demos, tutorial recordings, and quick screen captures where desktop clutter would distract from the work.

Known V1 limitation
