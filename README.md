# Pocket Orchard

Small, private apps for everyday life.

This public site hosts app pages, privacy policies, support pages, and practical guides for Pocket Orchard apps.

## Apps

- [CleanScreen Drop](https://zhaibin2018.github.io/pocket-orchard/apps/cleanscreen-drop/) - a free macOS menu bar app that hides desktop icons before screen sharing, demo recording, or calls.
- [Tiny Mood Garden](https://zhaibin2018.github.io/pocket-orchard/apps/tiny-mood-garden/) - a private daily mood ritual for iPhone and iPad.
